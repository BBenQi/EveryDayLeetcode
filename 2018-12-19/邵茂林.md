1.用栈实现队列<br/>
思路：使用两个栈，stackA和stackB，分别表示队头和队尾，<br/>
进队时将元素放入stackA中即可，出队时从stackB中出队，如果stackB为空的话则将stackA中的元素复制到stackB<br/>

```
class MyQueue {
    private Stack<Integer> stackA = null;
    private Stack<Integer> stackB = null;
    
    /** Initialize your data structure here. */
    public MyQueue() {
         stackA = new Stack<>();
         stackB = new Stack<>();
    }
    
    /** Push element x to the back of queue. */
    public void push(int x) {
        stackA.push(x);
    }
    
    /** Removes the element from in front of queue and returns that element. */
    public int pop() {
        if (stackB.empty()) {
            while(!stackA.empty()) {
                int temp = stackA.peek();
                stackB.push(temp);
                stackA.pop();
            }
            int ret = stackB.peek();
            stackB.pop();
            return ret;
        } else {
            int ret = stackB.peek();
            stackB.pop();
            return ret;
        }
        
    }
    
    /** Get the front element. */
    public int peek() {
        if(!stackB.empty()) {
            int ret = stackB.peek();
            return ret;
        } if (!stackA.empty()) {
            while(!stackA.empty()){
               int temp = stackA.peek();
                stackB.push(temp);
                stackA.pop();
            }
            
            return stackB.peek();
        }
        return 0;  
    }
    
    /** Returns whether the queue is empty. */
    public boolean empty() {
        return stackA.empty() && stackB.empty();
    }
}

 * MyQueue obj = new MyQueue();
 * obj.push(x);
 * int param_2 = obj.pop();
 * int param_3 = obj.peek();
 * boolean param_4 = obj.empty();
 */
```

 2.用队列实现栈</br>
 思路：队列的存储和栈是相反的,用两个队列来实现，入栈数据进入不为空的队列，如果都为空则进入queueA，<br>
 出栈时，如果queueA不为空，则将queueA中除最后一个元素数据复制到queueB中，弹出queueA中的最后一个元素即为栈顶元素<br/>
 
 ```
 class MyStack {
    
    private Queue<Integer> queueA = null;
    private Queue<Integer> queueB = null;

    public MyStack() {
        queueA = new LinkedList<>();
        queueB = new LinkedList<>();
    }

    /** Push element x onto stack. */
    public void push(int x) {
        if (!queueA.isEmpty())
            queueA.offer(x);

        if (!queueB.isEmpty())
            queueB.offer(x);

        if (queueA.isEmpty() && queueB.isEmpty())
            queueA.offer(x);
    }

    /** Removes the element on top of the stack and returns that element. */
    public int pop() {
        if (!queueA.isEmpty()) {
            while (queueA.size() > 1) {
                queueB.offer(queueA.poll());
            }
            return queueA.poll();
        } else {
            while (queueB.size() > 1) {
                queueA.offer(queueB.poll());
            }
           return queueB.poll();
        }

    }

    /** Get the top element. */
    public int top() {
        int top = 0;
        if (!queueA.isEmpty()) {
            while (queueA.size() > 1) {
                queueB.offer(queueA.poll());
            }
            top = queueA.peek();
            queueB.offer(queueA.poll());
        } else {
            while (queueB.size() > 1) {
                queueA.offer(queueB.poll());
            }
            top = queueB.peek();
            queueA.offer(queueB.poll());
        }
        return top;

    }

    /** Returns whether the stack is empty. */
    public boolean empty() {
        return queueA.isEmpty() && queueB.isEmpty();

    }
}

 * MyStack obj = new MyStack();
 * obj.push(x);
 * int param_2 = obj.pop();
 * int param_3 = obj.top();
 * boolean param_4 = obj.empty();
 */
 ```
